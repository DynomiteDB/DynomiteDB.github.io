Dynomite v0.5.7

Released on March 2, 2016

With the release of Dynomite v0.5.7 you can now run Dynomite on Ubuntu, RHEL and CentOS. New functionality includes the ability to differentiate between local and remote statistics, support for the Travis test framework, plus numerous bug fixes.
